# AI Toolbox Directory

A purely static web directory built with Vite, React, and TypeScript. This repository showcases various AI Apps, Companies, and Models.

## Local Development

```bash
# Run the dev server
npm run dev
```

## Deployment

Since this project has no backend and outputs static HTML/CSS/JS files, deploying to GitHub Pages is entirely straightforward.

```bash
# Deploy to GitHub Pages
npm run deploy
```
